SVM

Support vector machines (SVMs) are a set of supervised learning methods used for classification, regression and outliers detection.
Support Vector Machines: SVM is binary classification algorithm. Given a set of points of 2 types in N dimensional place,
SVM generates a (N — 1) dimensional hyperplane to separate those points into 2 groups. Say you have some points of 2 types in a paper which are linearly separable.
SVM will find a straight line which separates those points into 2 types and situated as far as possible from all those points.

  • ad:

    • effective in high dimensional spaces
    • effective in cases where number of dimensions
      • greater than the number of samples
    • subset of training points more memory efficient
    • versatile
      • different kernel functions specified for the decistion function

  • dis:

    • number of features greater than number of samples
      • overfitting in choosing kernel functions
    • not directly provide probability estimates
      • five fold cross validation expensive

Principal Component Analysis: PCA is a statistical procedure that uses an orthogonal transformation to convert a set of observations of possibly correlated variables into a set of values of linearly uncorrelated variables called principal components.
